Our Finall Project:
- What is the main idea of the web app?

Our main idea is to have a website of a traveling agency, 
one agent will specilize in transportation and how to move around, the other will specilize in recomendations and tourists places.

- What responsibility does each agent have?

Agent number 1 is responsible for transportation.
Agent number 2 is responsible for recomendations and planing the touristic places.

- How does the user move through the application from beginning to end?

1. Sign up / in
2. A page to pick the starter agent
3. A texting page of the agent and the user
* Maybe a function of image genorator / useable camera for questions

- What is the final product?

A website which will help families whoo struggle with planing their trip and need some assistence.
